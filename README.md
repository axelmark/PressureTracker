# Pressure Tracker

An application for tracking blood pressure. It allows you to record measurements, view charts, export data, and set reminders.

---

## 📱 Key Features

- **Add Measurements**  
  Systolic and diastolic pressure, pulse, note, medication intake flag.

- **Measurement History**  
  List with swipe-to-delete, expandable notes, color‑coded pressure categories.

- **Charts**  
  View trends for systolic, diastolic, and pulse over a week, month, or year.

- **Export Data**  
  Export to CSV and PDF (automatically saved to Documents/PressureTracker).

- **Backup & Restore**  
  Create and restore database backups.

- **Reminders**  
  Daily or weekly notifications with customizable time and day.

- **User Profile**  
  Store name, age, gender, height, weight, and calculate BMI.

- **Dark Theme**  
  Full support for light and dark themes with adaptive Material Design 3 colors.

---

## 🛠 Technologies

- **Language:** Java (Android SDK)
- **Architecture:** MVVM (ViewModel, LiveData, Repository)
- **Database:** Room
- **Charts:** MPAndroidChart
- **UI:** Material Design 3 (Material Components for Android)
- **Notifications:** AlarmManager, NotificationManager
- **Export:** CSV, PDF using PdfDocument
